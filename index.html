DESICOS

DESICOS is an European Project studying the Knock-Down Factors used in the Space Industry for te design of cylindrical and conical structures (rocket science...).

www.desicos.eu

This repository contains the plug-ins used along the project for the studies comparing geometric imperfections, load imperfections, stochastic analysis and so on.

See the AUTHORS file for the list of authors.

1.1 Abaqus plug-ins

The user-friendly way to use the plug-in is through the graphic interface. Just execute the file START_GUI.bat. You need Abaqus to be executable by the command line abaqus cae for this to work.

You can access all and more fuctionalities from the Python IDE in Abaqus::

>>> import sys
>>> sys.path.append(r'...\abaqus-conecyl-python')
>>> from conecyl import ConeCyl
>>> cc = ConeCyl()
>>> cc.fromDB('huehne_2008_z07')
>>> cc.create_model()

See more examples of how to access the Python functionalities in the folder studies.
